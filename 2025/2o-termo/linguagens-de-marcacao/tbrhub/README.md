# TBRHub - Biblioteca Digital 📚🤖
Para o TCS de LiMa (Linguagens de Marcação), que envolve criar um site que preze pela sustentabilidade, de tema qualquer. No caso, uma livraria. 📖

Seguindo a premissa de uma “enorme estante de livros”, TBR significa “To Be Read” e Hub, traduzindo a ideia para o português seria: Ponto a ser lido. O usuário poderá navegar livremente entre livros virtuais para ler, favoritá-los e até mesmo publicar sua própria obra (com aprovação de nossa equipe, para filtrar conteúdos indesejados).

**Tópicos 📝**
- **Tema:** Armazém de livros digital (como se fosse Netflix de livros);
- **Nome do Grupo:** Bibliotechs;
- **Integrantes:** Pedro Cordeiro, Eduardo, João Gritti, Filipe, Alice, Ana Luiza e Guilherme;
- **Breve descrição:** Condizente com plataformas de streaming como Netflix, neste caso, para livros, nós do site [tbrhub.com.br] (estamos a pensar no domínio ainda) procuraremos contatar editoras para permitir suas respectivas autorizações. 


**Problema: Acesso difícil à E-books. ❗❗**

_"Dado às paywalls excessivas dos renomados bancos de livros online, a intenção de nosso site é mitigar esses efeitos, buscando contatar com editoras reais que possuem o direito à propriedade intelectual para hospedarmos esse conteúdo no site sem problemas judiciais. Caso nossas tentativas de contatação sejam em vão, vamos, no mínimo permitir que usuários publiquem sinopses de livros com capa, nome do autor e etc, que será exibida somente com permissão dos administradores (no caso, o grupo Bibliotechs)."_

**Sustentabilidade 🌎🌲🍃**
- Fazer um cálculo para quando você finaliza a leitura de quantas árvores foram poupadas no processo (em relação à quantidade de páginas x papéis); 
- Mencionar locais que fazem trocas ou doações de livros, para evitar a incineração ou descarte incorreto;
