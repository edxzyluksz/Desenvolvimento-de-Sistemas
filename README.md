# 💻 Desenvolvimento de Sistemas - SENAI

Técnico Integrado ao Ensino Médio, realizado às terças e quintas, com a duração de janeiro de 2025 até dezembro de 2026. Seu objetivo principal é fornecer aos estudantes experiência para lidar com aplicações reais do mercado, mais especificamente em sistemas de informação, abordando diversas técnicas e ferramentas para produção.

Criado por EdxzyLuksz, este repositório tem como intuito registrar o máximo de experiências adquiridas e realizadas através de diversas situações de aprendizagem propostas pelos docentes.

## 🏷️ Nomenclaturas

O curso descreve o andamento dos semestres utilizando o sufixo 'termo', onde a divisão de matérias se encontra em pastas de `2025/` e `2026/` com esta palavra.

Para atividades avaliativas, a palavra 'somativa' é utilizada dentro da instituição, onde o estudante deve desenvolver uma solução para um problema de forma autônoma.

## 🛥️ Navegação Básica

- Demonstrações de conteúdos práticos (ex: Códigos de Java escritos pelo docente) --> `📁 demonstracoes/`
- Atividades autorais dado orientações dos professores --> `📁 atividades/`
- Avaliações de domínio realizadas (somativa) --> `📁 somativa/`

> OBS: Atualmente, está em fase de transição para separar o conteúdo interno destes diretórios como separados por linguagem de programação e frameworks.

## 👨‍🏫 Cursos Externos/Transversais

Se tratam de iniciativas da instituição SENAI para que seus alunos desenvolvam habilidades que excedam as exploradas do curso principal. Seus nomes e certificados de conclusão do autor podem ser encontrados em `📁 certificados/` de `2025/` ou `2026/`.
