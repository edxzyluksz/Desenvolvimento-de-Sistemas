O HTML (HyperText Markup Language) é a linguagem de marcação que define a estrutura e o conteúdo de uma página web. Ele é responsável por organizar elementos como textos, imagens, links, tabelas e formulários. Já o CSS (Cascading Style Sheets) é a linguagem de estilo que permite controlar a aparência desses elementos, definindo cores, fontes, layouts, espaçamentos e muito mais.
Juntos, HTML e CSS são a base de qualquer site que você visita. Dominá-los é o primeiro passo para se tornar um desenvolvedor web competente.
Nesta seção, você revisitará os conceitos fundamentais do HTML e CSS.

- Adicione um espaçamento interno (padding) e uma borda ao parágrafo da sua descrição.

Crie um novo conjunto de arquivos (produto.html e produto.css). Seu objetivo é criar um "cartão de produto" simples que poderia aparecer em um e-commerce.
No produto.html, crie uma estrutura básica contendo:
- Um título para o produto (`<h2>`).
- Uma imagem do produto.
- Um parágrafo com a descrição do produto.
- Um parágrafo com o preço.
- Um botão "Comprar".
- No produto.css, aplique os seguintes estilos:
- Dê uma largura e altura fixas ao "cartão" (use uma div para envolver todos os elementos do produto).
- Adicione uma borda e uma sombra suave ao cartão para destacá-lo.
- Centralize o cartão na página.
- Use padding interno para afastar o conteúdo das bordas do cartão.
- Alinhe o texto do título, descrição e preço ao centro.
- Estilize o botão "Comprar" com uma cor de fundo, cor do texto, padding e bordas arredondadas.
- Esta é a parte onde você aprende a criar layouts complexos e, mais importante, a tornar suas páginas web responsivas (adaptáveis a diferentes tamanhos de tela).

Crie um novo conjunto de arquivos (layout.html e layout.css). Seu objetivo é criar um layout web responsivo com as seguintes seções:
- Uma barra superior com um título e talvez alguns links de navegação.
- Uma área principal com duas ou três colunas de conteúdo (use Flexbox ou Grid para isso).
- Uma barra inferior com informações de copyright.
- Estruture seu layout.html usando tags semânticas (ex: `<header>, <main>, <footer>, <nav>, <section>, <article>`).
- Use Flexbox ou Grid para organizar o conteúdo do header (ex: título à esquerda, links à direita).
- Use Flexbox ou Grid para criar as colunas de conteúdo na seção main. As colunas devem ter larguras proporcionais.
- Estilize o footer com cor de fundo e texto centralizado.

Use para que, em telas menores (ex: largura máxima de 768px - tablets e celulares), as colunas da seção main se empilhem verticalmente (cada uma ocupando 100% da largura).
Ajuste o tamanho da fonte ou espaçamentos se necessário para telas menores.
