# Sistemas Operacionais
## O QUE É UM SISTEMA OPERACIONAL?

Nomes como Windows, Linux, iOS e Android são SOs.

Eles são softwares (instaláveis) que gerenciam os recursos e aplicativos de seu dispositivo, e cumprem o papel de garantir o bom funcionamento do aparelho.
É importante considerar que um SO esteja em harmonia com o hardware (peças como placa de vídeo e processadores) para melhor eficiência e uso confortável.
Todos os dispositivos inteligentes possuem um SO (incluindo TVs) e cada sistema possui uma interface e política de segurança únicas, podendo agradar ou não o usuário.

## TIPOS DE SISTEMAS OPERACIONAIS

SISTEMA OPERACIONAL
O Software Base, muitas vezes definido como Kernel, que controla todo o Hardware da máquina, portas de comunicação, periféricos, processadores, memórias, arquitetura de armazenamento, e toda a interação base com o usuário final.

LINGUAGENS E SISTEMAS DE ALTO NÍVEL
Recursos que se aproximam mais de nós humanos: como inteligências artificiais que possuem sentido para nós no literal. Uma máquina reproduzir um texto de "Olá" é considerado Alto Nível.

- SO DE PROPÓSITO GERAL
Compatíveis com diversos modelos de desktops e laptops. Permite o usuário executar múltiplas tarefas simultaneamente.
- SO MÓVEL
Atendem às necessidades de dispositivos móveis. Smartphones e tablets são menores e menos complexos.
- SO EMBARACADO
Feito para fins específicos, como em eletrodomésticos, aeronaves e linhas industriais.
- SO DE REDE (NOS - sigla em inglês)
Facilita a comunicação entre dispositivos em uma rede local. O conceito está sendo menos usado, mas é operado em dispositivos como roteadores e firewall.
- SO DE TEMPO REAL
Deve operar sem atrasos ou falhas de carregamentos, pois deve interagir com o mundo real de maneira constante e imediata, usados em indústrias da saúde, robótica, gás e petróleo.
